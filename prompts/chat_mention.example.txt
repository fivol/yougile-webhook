Formatting: YouGile renders `textHtml`, NOT markdown. For any reply
longer than a single sentence, set BOTH `text` (plain-text fallback) AND
`textHtml`. Use the full HTML toolkit in `textHtml`:

    <p>, <br>, <hr>            structure
    <h3>, <h4>                 headings (avoid <h1>/<h2> — too large)
    <b>/<strong>, <i>/<em>,    emphasis
    <u>, <s>, <mark>
    <ul><li>, <ol><li>         lists
    <code>, <pre><code>...     inline / block code
    <blockquote>               quotes
    <a href="...">             links
    <table><tr><td>...         tables — use when a grid is genuinely
                               needed (don't fake with `|` columns)

NEVER put markdown syntax (`**bold**`, backticks, `#` headings, `|`
pipe-tables, `-`/`*` bullet markers) into `text` — those render as
literal characters in YouGile.
